# Sticky Notes - 付箋メモアプリ

シンプルで美しい付箋メモアプリケーションです。Apple風のモダンなデザインで、直感的に使えます。

## 機能

### 基本機能
- **付箋の作成・編集・削除** - ワンクリックで新しい付箋を追加し、クリックで編集
- **ドラッグ＆ドロップ** - 付箋を自由にドラッグして好きな位置に配置
- **8色のカラーバリエーション** - サンシャイン、コーラル、ラベンダー、ミント、スカイ、ピーチ、ローズ、セージ

### 便利な機能
- **重要な付箋を固定** - 最大3つまで付箋を固定して目立たせることが可能
- **データ永続化** - ブラウザをリロードしても付箋の内容と位置が保存される（localStorage使用）

### デザイン
- **レスポンシブデザイン** - デスクトップ・タブレット・スマートフォンに対応
- **Glassmorphismスタイル** - 半透明でぼかし効果のあるモダンなデザイン
- **スムーズなアニメーション** - ホバー、ドラッグ時の視覚的フィードバック

## 技術スタック

- **フレームワーク**: React 19
- **言語**: TypeScript
- **ビルドツール**: Vite 7
- **スタイリング**: Tailwind CSS 4.x
- **UIコンポーネント**: shadcn/ui
- **アイコン**: lucide-react
- **テスト**: Vitest + Testing Library

## 使い方

1. **付箋を作成**: ヘッダーの「新規作成」ボタンまたは空状態の「最初の付箋を作成」ボタンをクリック
2. **付箋を編集**: 付箋の本文部分をクリックして編集モードに入り、テキストを入力
3. **付箋を移動**: 付箋をドラッグして好きな位置に配置
4. **色を変更**: パレットアイコンをクリックしてカラーピッカーから色を選択
5. **付箋を固定**: ピンアイコンをクリックして固定（最大3つまで）
6. **付箋を削除**: ゴミ箱アイコンをクリック

## デザインシステム

### カラーパレット

| 名前 | 色 | 用途 |
|------|------|------|
| サンシャイン | 黄色系 | デフォルト、アイデアメモ |
| コーラル | オレンジ系 | 緊急タスク |
| ラベンダー | 紫系 | クリエイティブ |
| ミント | エメラルド系 | 完了タスク |
| スカイ | 青系 | 情報メモ |
| ピーチ | ローズ系 | 個人メモ |
| ローズ | ピンク系 | 重要事項 |
| セージ | 緑系 | 日常タスク |

### アニメーション

- **ホバー時**: スケール1.02倍、シャドウ強調
- **ドラッグ時**: スケール1.05倍、シャドウ最大
- **ボタンホバー**: スケール1.1倍、回転アニメーション
