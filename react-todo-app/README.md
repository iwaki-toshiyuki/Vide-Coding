# Todo アプリ

シンプルでモダンなTodoアプリケーションです。一画面でタスクの作成、編集、削除、並び替えが可能です。

## 機能

- **タスクの作成**: フォームからテキストを入力して新しいタスクを追加
- **タスクの編集**: タスクをダブルクリックして内容を編集
- **タスクの削除**: 削除ボタンでタスクを削除
- **タスクの並び替え**: ドラッグ&ドロップでタスクの順序を変更
- **完了状態の管理**: チェックボックスでタスクの完了/未完了を切り替え
- **データの永続化**: localStorageを使用してブラウザにデータを保存

## 技術スタック

- **React 19**: UIフレームワーク
- **Vite**: ビルドツール
- **Framer Motion**: アニメーションライブラリ
- **dnd-kit**: ドラッグ&ドロップライブラリ

## 開始方法

### 前提条件

- Node.js 18以上

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。

### 本番用ビルド

```bash
npm run build
```

### ビルドのプレビュー

```bash
npm run preview
```

## データ構造

```javascript
{
  id: string,         // ユニークID (UUID)
  text: string,       // タスク内容
  completed: boolean, // 完了フラグ

}
```

## 操作方法

| 操作 | 説明 |
|------|------|
| テキスト入力 + 追加ボタン / Enter | 新しいタスクを追加 |
| チェックボックスをクリック | 完了/未完了の切り替え |
| タスクをダブルクリック | 編集モードに切り替え |
| 編集中にEnter | 編集を確定 |
| 編集中にEscape | 編集をキャンセル |
| ドラッグハンドルをドラッグ | タスクの並び替え |
| ×ボタンをクリック | タスクを削除 |
